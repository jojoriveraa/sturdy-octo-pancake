    public class ClassGeneratingEntityInstantiator
extends Object
implements EntityInstantiator
    An EntityInstantiator that can generate byte code to speed-up dynamic object instantiation. Uses the PersistentEntity's PreferredConstructor to instantiate an instance of the entity by dynamically generating factory methods with appropriate constructor invocations via ASM. If we cannot generate byte code for a type, we gracefully fall-back to the ReflectionEntityInstantiator.
